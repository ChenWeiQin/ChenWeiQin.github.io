学习位运算的要点

  1. 什么是位运算?

    计算机最小的存储单位是字节,一个字节占8位,所谓的位运算就是对各个字节上每一个位进行运算操作. 位运算的特点是高效,如不需要第三个变量就可以实现变量交换,快速判断奇偶数

  2. 学习位运算所需知识点

    1. 什么是二进制?

      逢二进一,只有两个数字0和1,这个做的原因信号稳定,如0表示不通电,1表示通电.

    2. 什么是原码,反码,补码?

      原码: 就是使用二进制来表示 unsinged int num =5; int 占4个字节,一个字节占8 位.则用二进制表示如下: 00000000 00000000 00000000 00000101 这就是5的原码

      反码: 一个无符号数(即正数),它的原码,反码,补码都一样的.对于有符号(即负数)的反码=符号位不变(最高位表示符号位,所谓符号位就是标识它是正数还是负数)对原码取反(1变0,0变1)

      singed int num = -5;

      -5原码: 10000000 00000000 00000000 00000101

      -5原码: 1111111 11111111 11111111 11111010 就是把原码位对应的位置上把1->0,0->1

      补码:负数的补码=它的反码 + 1;

      -5原码: 1111111 11111111 11111111 11111011

    3. 计算机在运算过程是以什么进行运算?

      都是以补码方式进行运算的

    4. 运算后的结果是补码,然后把它推成原码方式,这样才是最后的计算的结果

    5. 0的原码,反码,补码都是0

    6. 在php语言中所有的数值都有符号的

  3. 位逻辑运算符规则

    $a & $b And（按位与） 将把 $a 和 $b 中都为 1 的位设为 1。(11=>1)

    $a | $b Or（按位或） 将把 $a 和 $b 中任何一个为 1 的位设为 1。(10 =>1 11=>1 01=>1)

    ~ $a Not（按位取反）将 $a 中为 0 的位设为 1，反之亦然。

    $a ^ $b Xor（按位异或）将把 $a 和 $b 中两个位上相同为0不同为1。(10 =>1 11=>0 01=>1)

  4. 位移运算符规则

    怎么区分是左移还是右移，看箭头方向。
    $a << $b Shift left（按位左移）将 $a 中的位向左移动 $b 次（每一次移动都表示”乘以 2”）。
    可以理解为：符号位不变，低位补0

    $a >> $b Shift right（按位右移）将 $a 中的位向右移动 $b 次（每一次移动都表示”除2”）。
    可以理解为：符号位不位，低位溢出，并用符号位补高位

    对右移运算的结果：如果是小于0 直接取结果为0，如果大于0只取结果中整数部分。

使用案例

正数的原码,反码,补码都一样的, 如果运算的结果是负数则把它从补码->(-1)反码->(1->0,0->1)原码

练习题：
-3&2 结果是：0
-3|-6 结果是：-1
-3^-6 结果是：8
-3^6 结果是：-5
~-3 结果是：2
7 >> 2 结果是：1 （7/2/2）
7 << 3 结果是：56 (7 * 2^3)
7 >> 3 结果是：0 (7/2/2/2 )
对右移运算的结果：如果是小于0 直接取结果为0，如果大于0只取结果中整数部分。

-3 原码：10000000 00000000 00000000 00000011 （12^0 + 12^1）0 剩以任何数得0
-3 反码：11111111 11111111 11111111 11111100
-3 补码：11111111 11111111 11111111 11111101
2 补码：00000000 00000000 00000000 00000010
得到两个数的补码，就可以开始运算了
-3&2 的结果补码是：
00000000 00000000 00000000 00000000 (看它的符号位是0 表示正数，正数的原码 补码 反码都一样)

11111111 11111111 11111111 11111110 =》 原码（1-》0 0-》1）
10000000 00000000 00000000 00000001
-3^-6后的补码：
00000000 00000000 00000000 00000111(看它的符号位是 0 表示是正数，那么正的补码 反码 原码都一样)
